# 🍴 Forkify

A modern recipe search and bookmarking web application built using HTML, CSS, and JavaScript — as part of the **Complete JavaScript Course** by Jonas Schmedtmann.

## 🚀 Features

- 🔍 **Search recipes** from a public API
- 📄 **View detailed recipe instructions**
- 🔢 **Adjust servings** and ingredients dynamically
- 🔖 **Bookmark your favorite recipes** (persists across sessions)
- 📝 **Upload your own recipes**
- ⚡ **Responsive UI** with modern styling and smooth loading spinners

## 🧠 What I Learned

- ES6+ JavaScript (modules, classes, async/await)
- MVC architecture in front-end applications
- Webpack & Parcel for modern development workflow
- Fetch API and error handling
- Local storage and state management
- DOM manipulation and event delegation
- User experience optimization with loading spinners and error messages

## 🛠️ Tech Stack

- HTML5
- CSS3 (with custom styling)
- JavaScript (ES6+)
- Parcel Bundler
- forkify API (`https://forkify-api.herokuapp.com/v2`)
